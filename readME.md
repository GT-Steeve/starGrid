# 🎮 Color Tap

**Color Tap** est un petit jeu web interactif développé en **HTML, CSS et JavaScript**. Le but du jeu est de cliquer sur les cases d’une grille 5x5 pour découvrir une étoile cachée. Chaque case change de couleur à chaque clic et un compteur de clics est affiché. Le jeu garde la progression même après un rechargement grâce au `localStorage`.

---

## 📝 Fonctionnalités principales

- **Grille 5x5** responsive et mobile-friendly  
- **Couleurs aléatoires agréables** et transitions fluides  
- **Compteur individuel** sur chaque case et **compteur global**  
- **Mode sombre et clair** activable par un bouton  
- **Étoile cachée** :
  - Placée aléatoirement sur une case
  - Débloquée après un certain nombre de clics sur cette case
  - Nombre de clics requis dépend du mode de difficulté :
    - Facile : 2 clics  
    - Moyen : 4 à 6 clics  
    - Doom : 2 à 10 clics  
- **Animations** : effet spécial lors de la découverte de l’étoile et particules dorées  
- **Pop-up de victoire** :
  - Affiche le nombre de clics pour trouver l’étoile
  - Boutons pour **Rejouer** ou **Voir la grille**
  - Bouton pour **Revoir le score** après fermeture du pop-up
- **Sauvegarde automatique** :
  - Couleurs des cases
  - Compteurs individuels et total
  - Étoile trouvée ou non
  - Meilleur score

---

## ⚡ Technologie

- HTML5  
- CSS3 avec animations et transitions  
- JavaScript ES6  
- Utilisation de `localStorage` pour sauvegarde persistante
